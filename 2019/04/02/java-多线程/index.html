java-多线程

2019-04-02

进程和线程

  1. 进程是资源分配的最小单位，线程是cpu调度的最小单位

  2. 进程是一个独立应用，拥有独立的地址空间

  3. 进程切换开销大

当一个线程执行出错后，所属的进程就死了

Java的进程线程

对操作系统的功能进行了封装

特点:

  1. 一个程序会产生一个进程，一个程序至少包含一个线程
  2. 一个进程对应一个jvm实例，多个线程共享jvm的堆
  3. 程序自动创建主线程
  4. jvm中还有其它线程如GC等
